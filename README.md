# Google Reviews ABSA

An NLP project for building an Aspect-Based Sentiment Analysis (ABSA) dataset from Google Places reviews of Bakso Pajero Colombo.

## Features
- Google reviews data collection
- Text preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Aspect-based sentiment analysis preparation
- JSONL annotation dataset creation
- Streamlit application

## Files
- `absa_dataset_analysis.ipynb` → Jupyter Notebook for EDA and preprocessing
- `app.py` → Streamlit application
- `absa_annotations.jsonl` → annotated ABSA dataset
- `google_reviews_dataset.csv` → review dataset
- `requirements.txt` → project dependencies

## Technologies
- Python
- Pandas
- NLP
- Streamlit
- Jupyter Notebook

## Description
This project focuses on building an Aspect-Based Sentiment Analysis dataset using Google Places reviews from Bakso Pajero Colombo for natural language processing research and analysis.
